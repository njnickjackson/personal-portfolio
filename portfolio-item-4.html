ChemistryWordle

HTML, CSS, & JavaScript

GitHub Repo Live version

In this word game, the user gets six guesses to find the correct word. If the user guesses a letter that is in the word but is in the incorrect spot, the letter will be yellow. Correct placement will be green. Try it out and showcase your skills!

Wordle is my favorite game to play in the morning before work. I had a blast recreating this game with a chemistry twist. I actually use this game in my chemistry classes for extra credit!

This project was built with HTML, CSS, and JavaScript. During my recreation of Wordle, I implemented some very fun CSS animations and JavaScript tricks to make this game have a smooth user experience!
