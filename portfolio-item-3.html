E-commerceStore

React & Next.js

GitHub Repo

I have created an e-commerce website where users can view and purchase electronics! When users add items to their carts, they can proceed to checkout and actually purchase them! Users are prompted to enter payment information and can submit purchase orders.

This application implements React.js along with the “Stripe API.” I have connected this application to the Stripe API so that users can purchase items with a fast and secure service. After users proceed to checkout, they are prompted to purchase their items via the Stripe API.

I had an absolute blast creating this app with React and Next.js! This app has turned into a fully-functional e-commerce website where users can actually purchase electronics!
